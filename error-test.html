🧪 テスト手順

1. ディープリンク機能テスト

  1. 上記の「医療システムからの遷移」リンクをクリック
  2. 青色の通知バナーが表示されることを確認
  3. 評価期間と従業員IDが自動入力されることを確認
  4. ブラウザコンソールでパラメータログを確認

2. 一時保存機能テスト

  1. V3異議申立ページを開く
  2. 申し立て理由を入力（10文字以上）
  3. 「下書き保存」ボタンをクリック
  4. ページをリロード
  5. 「下書き読み込み」ボタンで復元を確認
  6. 30秒待機して自動保存を確認

3. エラーハンドリングテスト

  1. MCPサーバーを停止: cd mcp-integration-server && npm stop
  2. V3異議申立フォームを送信
  3. ネットワークエラーメッセージが表示されることを確認
  4. 「再試行」ボタンが表示されることを確認
  5. 下書きが自動保存されることを確認

4. リトライ機能テスト

  1. ブラウザの開発者ツールでネットワークを「Slow 3G」に設定
  2. フォームを送信
  3. コンソールで「リトライ 1/3...」メッセージを確認
  4. 指数バックオフ（1秒→2秒→4秒）の動作を確認

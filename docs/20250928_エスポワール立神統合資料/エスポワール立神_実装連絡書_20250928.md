## 3. システム連携仕様

### 3.1 API連携

#### 権限レベル取得API
```http
POST /api/v1/calculate-level
```

**リクエスト例**:
```json
{
  "staffId": "ESP_001",
  "facilityId": "espoir_tategami"
}
```

**レスポンス例**:
```json
{
  "staffId": "ESP_001",
  "facilityId": "espoir_tategami",
  "staffName": "石崎直樹",
  "position": "施設長",
  "accountLevel": 13,
  "breakdown": {
    "baseLevel": 13,
    "experienceBonus": 0,
    "leaderBonus": 0,
    "facilityAdjustment": 0
  }
}
```

### 3.2 組織構造の特記事項

1. **フロア体制**
   - 介護部はA/B/Cの3フロア体制
   - 各フロアに主任とフロアマネージャーを配置

2. **兼任ポジション**
   - 入所課課長が支援相談室長を兼任
   - 在宅課課長が居宅介護支援事業所管理者を兼任

3. **事業所構成**
   - 通所リハビリテーション事業所
   - 訪問リハビリテーション事業所
   - 訪問介護事業所
   - 居宅介護支援事業所
